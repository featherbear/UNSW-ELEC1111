Nodal Analysis

 

 

A branch - single element

Node - point of connection between branches

Based on KCL

 

Use node voltages (potential of each node) as the main circuit variables

Voltages are relative to reference nodes

 

Objective: To solve for node voltages

An -node circuit will need  voltages and  equations

 

Also requires the solution of a  system of equations.

 

KCL applied at each node except for one - the reference node

 

 

 

Reference node:

  • Can be any node
  • Commonly
    • Ground node
    • Top or bottom node
    • Node connected to the highest number of branches

 

 

Steps (circuit with  nodes)

  • Select a node as the reference node
  • Assign voltages , , …,  - Relative to the reference node
  • Apply KCL to each of the non-reference nodes
    • For resistors, use Ohm's law to express currents in terms of voltages - keep in mind the passive sign convention
  • Solve resulting  simultaneous equations
  • SIMPLIFY THE CIRCUIT FIRST

 

  • Choose ground as a reference node ()
