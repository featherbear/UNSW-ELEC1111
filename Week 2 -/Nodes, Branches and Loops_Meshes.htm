Circuit elements can be interconnected in multiply ways

 

A branch represents a single element such as a voltage source or a resistor

A node is the point of connection such as a voltage source or a resistor

A loop is any closed path in a circuit

A mesh is a loop that contains no other loop

 

  • Insert diagram

 

 

 

 

Two or more elements are in series if they exclusively share a single node, and consequently carry the same current

 

Two or more elements in parallel if they are connected to the same two nodes and consequently have the same voltage across them
