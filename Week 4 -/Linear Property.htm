Linear Property

Monday, 13 August 2018

12:15 PM

A resistor is a linear electrical element since the voltage-current relationship satisfies both the homogeneity and additivity properties

 

Homogeneity

If the input is multiplied by a constant, the output is also multiplied by the same constant

 

Additivity

The response to the sum of the inputs is the sum of the individual responses to each input

 

A linear circuit is one whose output is linearly related (or directly proportional) to its input
