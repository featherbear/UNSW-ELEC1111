Equivalent Circuit Model

Op Amps can be modelled with an input resistor R_i, and the output as a voltage-controlled voltage source with an output resistor

 

V_o = Av_d = A(v_2 - v_1)

 

R_o - thevenin equivalent resistor from output terminals

R_i - thevenin equivalent resistor from input terminals
