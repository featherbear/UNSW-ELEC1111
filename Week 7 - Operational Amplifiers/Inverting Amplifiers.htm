Inverting Amplifiers

Monday, 10 September 2018

12:44 PM

An inverting amplifier reverses the polarity of the input signal whilst amplifying it.

 

  • The noninverting input is grounded.
  • The inverting input is connected to the output via a feedback resistor,
  • The input voltage , is also connected to the inverting input via another resistor
